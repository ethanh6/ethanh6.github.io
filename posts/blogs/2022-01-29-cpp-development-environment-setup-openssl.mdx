Install OpenSSL in Ubuntu 20.04

```bash
$ sudo apt-get install libssl-dev
```

<br />

Test the installation
```bash
$ which openssl
/usr/bin/openssl   ===> this should be the path
$ openssl
OpenSSL>           ===> this should open a openssl dialogue 
```

<br />

Write a simple C++ file test.cpp to play with it

```cpp
#include <iostream>
#include <openssl/evp.h>
int main(int argc, char *argv[]){
        std::cout << "Hello OpenSSL" << std::endl;
        // load all SSL algorithm
        OpenSSL_add_all_algorithms();
        return 0;
}
```

<br />

Compile and test it:

```bash
$ g++ test.cpp -o test -lcrypto
$ ./test
Hello OpenSSL         ===> this should show the message
```

<br />

Other OpenSSL Commands

```bash
OpenSSL> version
```
